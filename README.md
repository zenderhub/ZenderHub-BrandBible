# ZenderHub · BrandBible

El sistema de marca de ZenderHub en un brandbook interactivo de un solo archivo.
Oscuro, cinético, con momentos de magenta que irrumpen sobre el negro como neón en un almacén.

## Ver el brand bible

- **En vivo:** https://zenderhub.github.io/ZenderHub-BrandBible/ *(se activa con GitHub Pages)*
- **Local:** abre `index.html` en cualquier navegador. No requiere build ni dependencias.

## Qué incluye

Fundamentos · Logo (con descargas SVG) · Color · Tipografía · Espaciado · Radios ·
Movimiento · Componentes en vivo · Voz & tono (bilingüe) · Imagen, fotografía y video
(incluye piezas para redes sociales: anatomía, ritmo vertical, consistencia del set y
checklist de verificación) · Accesibilidad · Tokens & gobernanza.

Está construido sobre los mismos tokens que documenta (dark-first `#0A0A0F`, gradiente
firma `245deg #A3086A→#009CC9`, radius 23px, Exo + Plus Jakarta Sans + JetBrains Mono).

## Estructura

```
index.html        El brandbook completo (HTML/CSS/JS autocontenido)
logos/            Logos oficiales en SVG (negativo, color, blanco + favicons)
photos/           Imágenes de referencia para la guía de fotografía
```

## Notas

- Las fotos de la sección de imagen son material de referencia de licencia libre
  (Pexels) con el grading de marca aplicado, para ilustrar el tratamiento. Sustitúyelas
  por fotografía propia de ZenderHub siguiendo las reglas de esa sección.
- Uso interno del equipo.
